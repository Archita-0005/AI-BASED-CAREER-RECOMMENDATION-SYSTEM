An AI-based Career Recommendation System using Naive Bayes predicts suitable career options based on a user’s skills, interests, and preferences. It uses the Naive Bayes algorithm, a probabilistic machine learning method, to analyze input data and calculate the likelihood of different careers. By training on a dataset of skills and corresponding job roles, the system classifies users into the most relevant career paths and provides accurate, personalized suggestions


How to run: a)install python b)Install libraries: pip install pandas streamlit scikit-learn c)Run: python Maincode.py
